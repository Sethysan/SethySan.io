About me

Hello, I'm Seth Rakich, a Software Developer with a passion for creating robust, user-friendly applications. After recently completing an intensive full-stack coding boot camp at Tech Elevator, I've built a strong foundation in Java, Spring Boot, PostgreSQL, JavaScript and front-end frameworks like Vue.js and Bootstrap. My projects include building dynamic web applications and integrating state management solutions to improve user experience.

My focus on developing efficient solutions has always been at the core of my work, a skill seamlessly transferred into software development— helping me approach coding challenges with a solutions-oriented mindset.

Outside of work, I'm passionate about gardening, spending time with my daughters, and exploring new music— whether it's playing guitar, drums, or adding new discoveries to my ever-evolving playlist. I'm always excited to connect with like-minded professionals in tech, discuss software development, and explore new opportunities.
